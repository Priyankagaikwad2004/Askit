# 🧠 AskIt – AI-Powered Q&A Platform

AskIt is a modern Question & Answer platform powered by Django (REST API) and React (Vite). It allows users to ask questions, get intelligent answers, and explore discussions in a clean and interactive UI.

---

## ✨ Features

- 🔍 Smart, flexible search across questions, descriptions, and tags
- 🧠 Optional AI integration for intelligent responses
- 🧱 Modular Django REST backend
- ⚡ Fast, modern React + Vite frontend
- 📸 Rich question/answer content with image support
- 🧪 Easy to extend and customize

---

## 🛠️ Tech Stack

| Frontend         | Backend            | Database   | Others             |
|------------------|--------------------|------------|---------------------|
| React + Vite     | Django REST API    | PostgreSQL | Axios, TailwindCSS  |
